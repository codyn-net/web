The codyn software is made freely available under the GPL license. Codyn provides pre-built packages for Ubuntu, OS X and Windows, as well as all sources. The development of codyn uses git and the git repositories are publicly accessible. # Packages Pre-built binary packages are available for Ubuntu, OS X (>= 10.6) and Windows (limited support). ## Ubuntu Precise i386/x86_64 codyn.net provides an Ubuntu repository for both 32 and 64 bits Ubuntu precise. The repository is located at . For more information on configuring the repository and installing the packages, please see: . After the repository is configured, install the codyn-all package (`sudo apt-get install codyn-all`) to install all the codyn software components. ## OS X (>= 10.6 Snow Leopard) All codyn software is available for OS X from a universal installer package. The latest available version of the installer is [Codyn-3.4.1.pkg](http://download.codyn.net/releases/mac/Codyn-3.4.1.pkg). The oldest supported OS X platform is Snow Leopard (10.6). This package provides all of the codyn software components available for OS X, including the library, studio and rawc. The codyn library for OS X will be installed as an OS X framework making it easy to develop with codyn on OS X. ## Windows The pre-compiled binaries of libcodyn for windows can be found at . Note that at this time, only libcodyn is supported on Windows. There are currently no binaries for the studio or rawc. Furthermore, some features of libcodyn are not available on Windows (most notably python, blas and lapack). # Sources The latest sources for all the codyn components can be found at . Previous released versions of codyn are also available at this location. # Development Codyn uses [git](http://git-scm.com/) for version control of all the software. All software can be easily accessed through git and can be browsed online at [http://git.codyn.net/](http://git.codyn.net/).
